# **Laporkeun aka laporkeun**

> Codebase for Laporkeun webapps, and act as extended verson for this [old laporkeun-client codebase](https://github.com/DrDhemm/laporkeun-client).

### How to do run?

- Clone the repo
- Do `npm i`
- Run with `npm run dev`
